# PONKAN

関東近郊の水族館・美術館・博物館・科学館をめぐる、自分用スタンプラリーwebアプリ。
行った施設をタップして「ポンッと押す」だけで、訪問日つきのスタンプが貯まります。

## 機能

- **スタンプ帳** — 46館(初期データ)の丸スタンプが並ぶ。カテゴリでフィルタ、施設の追加・削除も可能
- **施設詳細** — 訪問日・メモの記録、公式サイト / Googleマップへのリンク
- **マップ** — Leaflet + OpenStreetMapで訪問済み / 未訪問をピン表示
- **実績** — 「はじめてのポン」「全館制覇」などのバッジ
- **データ** — localStorageに保存。JSONでエクスポート / インポート
- **PWA** — スマホのホーム画面に追加してアプリとして使える

## 開発

```sh
pnpm install
pnpm dev      # 開発サーバー
pnpm build    # 型チェック + 本番ビルド
```

## メモ

- 施設の座標はおおよその値。初期データは `src/data/facilities.ts` で編集できます
- 記録データはブラウザごと(localStorage `ponkan:v1`)に保存されます
